Init Database with tables, stored procedures and basic actions.

1. Open the 'SQL Server Management Studio Express'.
2. Connect.
3. Select 'New Query'
4. Write - "CREATE DATABASE ASTDB;"
5. Select 'Parse' and after 'Execute'
6. Select 'New Query' and copy all InitDatabase.sql content to the query.
7. Select 'Parse' and after 'Execute'
